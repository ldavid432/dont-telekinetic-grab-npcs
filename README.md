# Don't Telegrab NPCs

Prevents you from casting telekinetic grab on NPCs so you can pick up your drops in peace.

---

You can allow specific NPCs to be telegrabbed if you want by editing the whitelist.

There are several toggles for pre-made whitelist NPCs:
- Maze Guardian (mage training arena)
- Sailing creature corpses
- Sailing lost crates
- Guard from King's Ransom

There is also a custom input which takes the names of any additional NPCs you'd like to whitelist
